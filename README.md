# Validador-contraseña-python
Proyecto simple en Python de un validador de contraseñas utilizando reglas básicas de seguridad.

# Funcionalidades
* Verifica que la contraseña tenga entre 8 y 16 caracteres.
* Verifica que contenga al menos una letra mayúscula.
* Verifica que contenga al menos un carácter especial.

# Tecnologías utilizadas
* Python
* Funciones
* Bucles
* Condicionales

# Objetivo del proyecto

Practicar lógica de programación, modularización del código y validación de entradas en Python.

# Cómo ejecutarlo
* Tener Python instalado.
* Descargar el archivo .py
* Ejecutar en terminal:
python nombre_del_archivo.py
